How to pass data between React components

Ideally its' best to have a state management system to share data between react components. On the other hand smaller applications can work by components passing data among themselves.


In this blog I will explain three different ways to pass data between react components :


1) Passing data from parent to child component.
2) Passing data from child to parent component.
3) Passing data between sibling components.


Passing data from parent to child components:

We can pass data from parent to child components using props. Props are short for properties. If we want to send multiple variables from parent to child we can pass multiple props. 

                              
                                class Parent extends React.Component {
                                  render() {
                                      return (
                                          <div>
                                          <p>fhfhfhfh</p>
                                          </div>
                                      );    
                                  }
                                }

In the above example the parent component is sending two props to the child component. The below code snippet shows how to access this data in child component : 

                              
                                class Child extends React.Component {
                                  render() {
                                      return (
                                          <div>
                                          <p>Username is {this.props.name}</p>
                                          <p>The user date of birth is {this.props.dob}</p>
                                          </div>
                                      );    
                                  }
                                }
Passing data from child to parent component :

If we want to send data from child to parent component then we send a callback function as a prop to the child component. The child component uses the callback function to send data back to the parent component. The below code example shows this : 

                              
                                class Parent extends React.Component {
                                  state = {name: ""};
                                  callBackFunction = (dataFromChild) => {
                                    this.setState({name: dataFromChild});		
                                  }
                                  render() {
                                      return (
                                          <div>
                                            <div>
                                              {this.state.name}
                                            </div>
                                          </div>
                                      );    
                                  }
                                }
                              
                            
                              
                                class Children extends React.Component {
                                  sendData = () => {
                                    this.props.parentCallBack("Bunny");			
                                  }
                                  render() {
                                      return (
                                          <div>
                                            Child Component
                                          </div>
                                      );    
                                  }
                                }

In the above code snippet you can see that the child component uses the callback function passed as prop to send data to the parent. The child component calls this callback function and sends data. You can save the data passed from the child component in state to the parent component if necessary.


Between Sibling React components:



To pass data between sibling components, the data needs to be sent to the parent component and then passed to the sibling component. Below code example show how to do this: 

                              
                                class Pareent extends React.Component {
                                  state ={name: ""};
                                  callBackFuntion = (dataFromOneOfTheSiblingComponent) => {
                                    this.setState({name: dataFromOneOfTheSiblingComponent});
                                  }
                                  render() {
                                      return (
                                          <div>
                                            
                                          </div>
                                      );    
                                  }
                                }
                                

                                class SiblingComponent1 extends React.Component {
                                  sendData = () => {
                                    this.props.parentCallBack("Bunny");			
                                  }
                                  render() {
                                    return(
                                      <div>
                                        Child Component      
                                      </div>
                                    );
                                  }
                                }
                                

                                class SiblingComponent2 extends React.Component {
                                  render() {
                                    return(
                                      <div>
                                        {this.props.name}    
                                      </div>
                                    );
                                  }
                                }

In the above example SiblingComponent1 sends data to the Parent react component. The Parent component sends that data to SiblingComponent2 as props. This is the way sibling components send data to each other.
